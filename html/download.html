How to run the HPV Simulator 2.0 as an app / program

Windows

You can download a portable (does not have to be installed) .exe file for use on Windows systems. This is the best option if you have a computer running Windows.
64 Bit: Click here to download a zip file of the game for WINDOWS 64 bit
32 Bit: Click here to download a zip file of the game for WINDOWS 32 bit
To run, download the zip file from the link above and unzip the contents into a folder on your computer where you want to keep it. Click on the file named "HPV.exe" to run it. If you have issues, make sure that the folder titled "HPV_Data" is in the same directory as "HPV.exe".

Macintosh

You can download an application for Macintosh, but there have been some issues running it. It is probably more advisable to play it online at the moment if it does not work. You have to allow third party applications in settings somewhere and then it may or may not work.
Universal: Click here to download a universal zip file of the game for Macintosh
64 Bit: Click here to download a zip file of the game for Macintosh 64 Bit
32 Bit: Click here to download a zip file of the game for Macintosh 32 Bit
To run, download the zip file from the link above and unzip the contents into a folder on your computer where you want to keep it. Click on the app icon to run it. It will either throw up an error message saying something about security, work or display some other message and not run. In the case that a message about security is displayed, you will have to allow the computer to run the app in settings. Otherwise maybe try one of the other versions for Macintosh in the list above.
